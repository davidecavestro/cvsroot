Preferenze utente

La finestra di impostazione delle preferenze utente permette di modificare il comportamento dell'applicazione in base alle necessità dell'utente stesso.

E' suddivisa in tre sezioni:

  • impostazioni interfaccia

  • impostazioni dati

  • altro

Impostazioni interfaccia

Permette di impostare:

  • il colore dello sfondo del desktop interno all'applicazione, tramite il pulsante contraddistinti dall'etichetta colore del desktop. Premendo il pulsante reset di fianco, viene ripristinato il core predefinito.

  • Abilitare/disabilitare l'avvertimento sonoro per le azioni di avviso, tramite la casella di spunta contrassegnata dall'etichetta beep eventi.

  • Abilitare/disabilitare l'auto caricamento del grafico ad anello relativo alla distribuzione del lavoro nel sotto albero,tramite la casella di spunta contrassegnata dall'etichetta auto caricamento grafico ad anello.



Impostazioni dati

Permette di impostare:

  • la directory contenente il database dell'applicazione

  • il nome del database



Tramite il pulsante inizializza database è possibile inizializzare le strutture che contengono i dati.

Questi dati sono essenziali per la gestione del database contenente i dati applicativi. Assicurarsi di impostare un percorso protetto da accesso di estranei, poiché la perdita dei dati contenuti in tale percorso comporta la perdita di tutti i dati relativi ai progetti e avanzamenti registrati. A seconda del valore che tali dati assumono per l'utente, è consigliabile effettuare eventualmente delle copie di backup del la directory contenente il database.



Altro

Permette di impostare:

  • la dimensione del buffer di log.
