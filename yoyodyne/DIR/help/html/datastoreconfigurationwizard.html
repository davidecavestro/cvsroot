Procedura guidata di configurazione datastore

Permette di impostare:

  • la directory contenente il database dell'applicazione

  • il nome del database



Qualora il datastore non esista ancora è possibile inizializzare le strutture che contengono i dati.

Questi dati sono essenziali per la gestione del database contenente i dati applicativi. Assicurarsi di impostare un percorso protetto da accesso di estranei, poiché la perdita dei dati contenuti in tale percorso comporta la perdita di tutti i dati relativi ai progetti e avanzamenti registrati. In funzione del valore che tali dati assumono per l'utente, è consigliabile effettuare eventualmente delle copie di backup della directory contenente il database.
