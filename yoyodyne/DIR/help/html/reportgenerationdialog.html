Finestra di generazione report

La finestra di generazione report permette di:

  • scegliere un tipo di report, tramite la cartella Report

  • impostare eventuali filtri sui dati da analizzare, tramite la cartella Filtri

  • impostare le proprie preferenze relative al formato di output, tramite la cartella Tipo output.



Segue la descrizione dettagliata delle tre cartelle.



Report

Permette di scegliere quale report generare tra quelli disponibili. In generale, a diverso tipo di report corrisponde una diversa logica di estrazione, aggregazione, e presentazione dei dati.

I tipi di report attualmente disponibili sono:

  • lista semplice degli avanzamenti: è un elenco degli avanzamenti appartenenti al sotto albero selezionato, o all'intero progetto.

  • avanzamenti cumulati: produce i dati aggregati relativi agli avanzamenti dell'ultima settimana per il nodo selezionato, distinguendo quelli direttamente associati al nodo da quelli derivanti dal sotto albero (ATTENZIONE: non funziona ancora correttamente).

Filtri

I filtri permettono di limitare la mole di dati estratta, in modo da aumentare la leggibilità e l'efficacia del risultato.

E possibile filtrare per:

  • nodo radice

  • data di inizio e data di fine (non applicabile a tutti i report).

Tipo output

Permette di decidere il formato del risultato.

E' possibile scegliere se stampare direttamente il report, visualizzarlo a video, oppure salvarlo su di un file nel formato scelto.

I formati supportati sono:

  • PDF

  • HTML

  • XML

  • CSV

  • XLS
